Section 2.1 Fields

We now begin the process of abstraction. We will do this in stages, beginning with the concept of a field. First, we need to formally define some familiar sets of numbers.

Definition 2.1.1.

The rational numbers, denoted by \(\Q\text{,}\) is the set

\begin{equation*} \Q = \setof{\frac{a}{b}}{a,b\in \Z, \ b\ne 0}\text{.} \end{equation*}

Recall that in elementary school, you learned that two fractions \(\frac{a}{b}, \frac{c}{d} \in \Q\) are equivalent if and only if \(ad=bc\text{.}\)

Activity 2.1.1.

Prove that our elementary school definition of equivalent fractions is an equivalence relation.

We likely have an intuitive idea of what is meant by \(\R\text{,}\) the set of real numbers. Defining \(\R\) rigorously is actually quite difficult, and will occupy a significant amount of time in a first course in real analysis. Thus, we will make use of your intuition. It is certainly the case that \(\N\subseteq \Z\subseteq \Q\subseteq \R\text{,}\) but \(\R\) also contains many irrational numbers, such as \(\pi\text{,}\) \(e\text{,}\) \(\sqrt{2}\text{,}\) and so on.

Out of \(\R\) we may build the complex numbers.

Definition 2.1.2.

The complex numbers consist of all expressions of the form \(a+bi\text{,}\) where \(a,b\in \R\) and \(i^2 = -1\text{.}\) Given \(z = a+bi\text{,}\) we say \(a\) is the real part of \(z\) and \(b\) is the imaginary part. The set of complex numbers is denoted \(\C\text{.}\)

Definition 2.1.3.

Let \(X\) be a nonempty set. A function \(\star : X \times X \to X\) is called a binary operation. If \(\star\) is a binary operation on \(X\text{,}\) we say that \(X\) is closed under the operation \(\star\). [Given \(a,b\in X\text{,}\) we usually write \(a\star b\) in place of \(\star(a,b)\text{.}\)]

Investigation 2.1.2.

Which of \(+, -, \cdot, \div\) are binary operations:

  1. on \(\R\text{?}\)

  2. on \(\Q\text{?}\)

  3. on \(\Z\text{?}\)

  4. on \(\N\text{?}\)

  5. on \(\C\text{?}\)

The hallmark of modern pure mathematics is the use of axioms. An axiom is essentially an unproved assertion of truth. Our use of axioms serves several purposes.

From a logical perspective, axioms help us avoid the problem of infinite regression (e.g., asking How do you know? over and over again). That is, axioms give us very clear starting points from which to make our deductions.

To that end, our first abstract algebraic structure captures and axiomatizes familiar behavior about how numbers can be combined to produce other numbers of the same type.

Definition 2.1.4.

A field is a nonempty set \(F\) with at least two elements and binary operations \(+\) and \(\cdot\text{,}\) satisfying the following field axioms:

  1. F1..

    Given any \(a,b,c\in F\text{,}\) \((a+b)+c = a+(b+c)\text{.}\)

  2. F2..

    Given any \(a,b\in F\text{,}\) \(a+b= b+a\text{.}\)

  3. F3..

    There exists an element \(0_F\in F\) such that for all \(a\in F\text{,}\) \(a+0_F = 0_F + a = a\text{.}\)

  4. F4..

    Given any \(a\in F\) there exists a \(b\in F\) such that \(a+b = b + a =0_F\text{.}\)

  5. F5..

    Given any \(a,b,c\in F\text{,}\) \((a\cdot b)\cdot c = a\cdot (b\cdot c)\text{.}\)

  6. F6..

    Given any \(a,b\in F\text{,}\) \(a\cdot b = b\cdot a\text{.}\)

  7. F7..

    There exists an element \(1_F\in F\) such that for all \(a\in F\text{,}\) \(1_F\cdot a = a\cdot 1_F = a\text{.}\)

  8. F8..

    For all \(a\in F\text{,}\) \(a\ne 0_F\text{,}\) there exists a \(b\in F\) such that \(a\cdot b = b\cdot a = 1_F\text{.}\)

  9. F9..

    For all \(a,b,c\in F\text{,}\) \(a\cdot (b+c) = a\cdot b + a\cdot c\text{.}\)

  10. F10..

    For all \(a,b,c\in F\text{,}\) \((a+b)\cdot c = a\cdot c + b\cdot c\text{.}\)

For brevity's sake, we will usually write \(a\cdot b\) as \(ab\text{.}\) Additionally, we will usually drop the subscripts on \(0,1\) unless we need to distinguish between fundamentally different identities in different fields.

Investigation 2.1.3.

Which of the following are fields? If no operations are specified, assume the most obvious operations (e.g., the addition and multiplication you have known for years). For most, a short justification or counterexample is sufficient.

  1. \(\N\)

  2. \(\Z\)

  3. \(2\Z\text{,}\) the set of even integers

  4. \(\Q\)

  5. \(\Z_{10}\)

  6. \(\Z_{11}\)

  7. \(\R\)

  8. \(\C\text{,}\) with addition defined as \((a_1 + b_1 i) + (a_2 + b_2 i) := (a_1 + a_2) + (b_1 + b_2)i\text{,}\) and multiplication defined as \((a_1 + b_1 i)(a_2 + b_2 i) := (a_1 a_2 - b_1 b_2) + (a_1 b_2 + b_1 a_2) i\text{.}\)

  9. \(\mathcal{M}_2(\R) := \setof{\left(\begin{matrix}a \amp b \\ c \amp d \end{matrix} \right)}{a,b,c,d\in\R}\) 1 , the set of \(2\times 2\) matrices with real coefficients using the usual definition of matrix multiplication 2  and matrix addition.

For students who have taken a linear algebra course.
Recall that, if \(\left(\begin{matrix}a_1 \amp b_1 \\ c_1 \amp d_1 \end{matrix} \right), \left(\begin{matrix}a_2 \amp b_2 \\ c_2 \amp d_2 \end{matrix} \right)\in \mathcal{M}_2(\R)\text{,}\) then \(\left(\begin{matrix}a_1 \amp b_1 \\ c_1 \amp d_1 \end{matrix} \right) \cdot \left(\begin{matrix}a_2 \amp b_2 \\ c_2 \amp d_2 \end{matrix} \right) = \left(\begin{matrix}a_1 a_2 + b_1 c_2 \amp a_1 b_2 + b_1 d_2 \\ c_1 a_2 + d_1 c_2 \amp c_1 b_2 + d_1 d_2 \end{matrix} \right)\text{.}\)
Solution

  1. \(\N\) is not closed under taking additive inverses, so is not a field.

  2. \(\Z\) is not a field, as there is no integer \(a\) such that \(2a = 1\text{.}\)

  3. \(2\Z\) is not a field for the same reason.

  4. \(\Q\) is a field.

  5. \(\Z_{10}\) is not a field; there is no \(\overline{a}\in \Z_{10}\) for which \(\overline{5}\cdot \overline{a} = \overline{1}\) (you can check them all; there are only 9 viable options).

  6. \(\Z_{11}\) is a field. The axioms pertaining to addition and multiplication were established earlier or rely on the same axioms holding for \(\Z\text{.}\)

  7. \(\R\) is a field

  8. \(\C\) is a field. Given \(a+bi\ne 0\text{,}\) \((a+bi)^{-1} = \frac{a}{a^2 + b^2} - \frac{b}{a^2 + b^2} i\text{.}\)

  9. \(\mathcal{M}_2(\R)\) is not a field. Matrix multiplication is not commutative.

In the Investigation 2.1.3, you determined which of sets of familiar mathematical objects are and are not fields. Notice that you have been working with fields for years and years and that our abstraction of language to that of fields is simply to allow us to explore the common features at the same time - it is inefficient to prove the same statement about every single field when we can prove it once and for all about fields in general.

Theorem 2.1.5. Properties of Fields.

Let \(F\) be a field.

  1. The additive identity \(0\) is unique.

  2. For all \(a\in F\text{,}\) \(a \cdot 0 = 0\cdot a = 0\text{.}\)

  3. Additive inverses are unique.

  4. The multiplicative identity \(1\) is unique.

  5. Multiplicative inverses are unique.

One consequence of Theorem 2.1.5 is that, given \(a\in F\text{,}\) \(b\in F\setminus \set{0}\text{,}\) we may refer to \(-a\) as the additive inverse of \(a\text{,}\) and \(b^{-1}\) as the multiplicative inverse of \(b\text{.}\) We will thus employ this familiar terminology henceforth.

Definition 2.1.6.

Let \(K\) be a field. If \(F\) is a nonempty subset satisfying

  1. \(F\subseteq K\) and

  2. \(F\) is a field under the same operations as \(K\text{,}\)

then we call \(F\) a subfield of \(K\text{,}\) and \(K\) an extension field of \(F\text{,}\) and say “\(K/F\) is a field extension.”

Many of the number sets with which you are familiar are fields.
Theorem 2.1.7.

The set of complex numbers \(\C\) is an extension field of \(\R\text{.}\)

Theorem 2.1.8.

For all primes \(p\text{,}\) \(\Z_p\) is a field.

Similarly, suppose \(\ideal{x}\subseteq \ideal{y}\text{.}\) Then \(x\in \ideal{y}\text{,}\) so \(y|x\text{.}\) Conversely, if \(y|x\text{,}\) then there is some \(r\in R\) such that \(x = yr\text{,}\) and thus for all \(ax\in \ideal{x}\text{,}\) \(ax = (ar)y \in \ideal{y}\text{.}\)

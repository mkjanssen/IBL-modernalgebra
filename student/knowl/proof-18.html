Proof

Let \(a\in \Z\text{.}\) Then \(m|a-a\text{,}\) so \(a \equiv_m a\text{.}\) Thus, \(\equiv_m\) is reflexive.

Let \(a,b\in \Z\) such that \(a\equiv_m b\text{.}\) This means that \(m|a-b\text{,}\) so there is some \(k\in \Z\) such that \(mk = a-b\text{.}\) Then \(m(-k) = b-a\text{,}\) so \(m|b-a\) and \(b \equiv_m a\text{.}\) Thus, \(\equiv_m\) is symmetric.

Finally, let \(a,b,c\in \Z\) such that \(a\equiv_m b\) and \(b\equiv_m c\text{.}\) Then \(m|a-b\) and \(m|b-c\text{,}\) so there are integers \(k_1, k_2\) such that \(m k_1 = a-b\) and \(m k_2 = b-c\text{.}\) Summing these equations yields \(m(k_1 + k_2) = (a-b) + (b-c) = a-c\text{,}\) so \(m|a-c\) and \(a\equiv_m c\text{.}\)

Definition 2.2.3.

A ring \(R\) is a nonempty set, together with binary operations \(+\) and \(\cdot\text{,}\) denoted \((R,+,\cdot)\text{,}\) and satisfying the following axioms.

  1. Given any \(a,b,c\in R\text{,}\) \((a+b)+c = a+(b+c)\text{.}\)

  2. Given any \(a,b\in R\text{,}\) \(a+b= b+a\text{.}\)

  3. There exists an element \(0_R\in R\) such that for all \(a\in R\text{,}\) \(a+0_R = 0_R + a = a\text{.}\)

  4. Given any \(a\in R\) there exists a \(b\in R\) such that \(a+b = b + a =0_R\text{.}\)

  5. Given any \(a,b,c\in R\text{,}\) \((a\cdot b)\cdot c = a\cdot (b\cdot c)\text{.}\)

  6. For all \(a,b,c\in R\text{,}\) \(a\cdot (b+c) = a\cdot b + a\cdot c\text{.}\)

  7. For all \(a,b,c\in R\text{,}\) \((a+b)\cdot c = a\cdot c + b\cdot c\text{.}\)

As with fields, when the ring \(R\) is clear from context, we will often write \(0\) in place of \(0_R\text{.}\)

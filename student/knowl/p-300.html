Let \(a,b\in R\) such that \(ab\) is a zero divisor. Then \(a,b\ne 0\) (else \(ab = 0\)). Since \(ab\) is a zero divisor, there is some \(c\ne 0\) such that \((ab) c = 0\text{.}\) If \(bc\ne 0\text{,}\) then \(a\) is a zero divisor, as \(a(bc) = 0\text{.}\) On the other hand, if \(bc=0\text{,}\) then \(b\) is a zero divisor, as \(b,c\ne 0\text{.}\)

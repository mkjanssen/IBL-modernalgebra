Activity 4.3.1.

Determine (with brief justification) whether \(a + I = b + I\) in the following rings \(R\text{.}\)

  1. \(a = 9\text{,}\) \(b = 3\text{,}\) \(I = \ideal{6}\text{,}\) \(R = \Z\)
  2. \(a = 10\text{,}\) \(b = 4\text{,}\) \(I = \ideal{7}\text{,}\) \(R = \Z\)
  3. \(a = 9\text{,}\) \(b = 3\text{,}\) \(I = \ideal{6}\text{,}\) \(R = \Z[x]\)
  4. \(a = x^2+x-2\text{,}\) \(b = x-1\text{,}\) \(I = \ideal{x+1}\text{,}\) \(R = \Q[x]\)
  5. (Challenge.) \(a = x^3\text{,}\) \(b = x^2+2x\text{,}\) \(I = \ideal{y-x^2, y-x-2}\text{,}\) \(R = \Q[x,y]\)

Definition 2.4.10.

Let \(R\) be an integral domain. We call \(R\) a Euclidean domain if there is a function \(\delta : R\setminus \set{0} \to \N_0\) such that:

This is the lowercase Greek letter delta.
}

  1. If \(a,b\in R\setminus \set{0}\text{,}\) then \(\delta(a) \le \delta(ab)\text{.}\)

  2. If \(a,b\in R\text{,}\) \(b\ne 0\text{,}\) then there exist \(q,r\in R\) such that \(a = bq+r\text{,}\) where either \(r = 0\) or \(\delta(r) \lt \delta(b)\text{.}\)

We call the function \(\delta\) a norm for \(R\text{.}\)

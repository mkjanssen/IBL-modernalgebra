Section 2.1 Fields

Guiding Questions.

In this section, we'll seek to answer the questions:

  • What are binary operations?

  • What is a field? What sorts of things can one do in a field?

  • What are examples of fields?

We now begin the process of abstraction. We will do this in stages, beginning with the concept of a field. First, we need to formally define some familiar sets of numbers.

Definition 2.1.1.

The rational numbers, denoted by \(\Q\text{,}\) is the set

\begin{equation*} \Q = \setof{\frac{a}{b}}{a,b\in \Z, \ b\ne 0}\text{.} \end{equation*}

Recall that in elementary school, you learned that two fractions \(\frac{a}{b}, \frac{c}{d} \in \Q\) are equivalent if and only if \(ad=bc\text{.}\)

Activity 2.1.2.

Prove that our elementary school definition of equivalent fractions is an equivalence relation. Recall Definition 1.4.2.

We likely have an intuitive idea of what is meant by \(\R\text{,}\) the set of real numbers. Defining \(\R\) rigorously is actually quite difficult, and occupies a significant amount of time in a first course in real analysis. Thus, we will make use of your intuition.

Out of \(\R\) we may build the complex numbers.

Definition 2.1.3.

The complex numbers consist of all expressions of the form \(a+bi\text{,}\) where \(a,b\in \R\) and \(i^2 = -1\text{.}\) Given \(z = a+bi\text{,}\) we say \(a\) is the real part of \(z\) and \(b\) is the imaginary part. The set of complex numbers is denoted \(\C\text{.}\)

As was mentioned in the Introduction, algebra comes from an Arabic word meaning “the reunion of broken parts”. We therefore need a way of combining two elements of a set into one; we turn to a particular type of function, known as a binary operation, to accomplish this.

Definition 2.1.4.

Let \(X\) be a nonempty set. A function \(\star : X \times X \to X\) is called a binary operation. If \(\star\) is a binary operation on \(X\text{,}\) we say that \(X\) is closed under the operation \(\star\). [Given \(a,b\in X\text{,}\) we usually write \(a\star b\) in place of the typical function notation, \(\star(a,b)\text{.}\)]

Investigation 2.1.5.

Which of \(+, -, \cdot, \div\) are binary operations:

  1. on \(\R\text{?}\)

  2. on \(\Q\text{?}\)

  3. on \(\Z\text{?}\)

  4. on \(\N\text{?}\)

  5. on \(\C\text{?}\) (Recall that for \(a_1 + b_1 i, a_2 + b_2 i \in \C\text{,}\) \((a_1 + b_1 i) + (a_2 + b_2 i) := (a_1 + a_2) + (b_1 + b_2)i\) and \((a_1 + b_1 i)(a_2 + b_2 i) := (a_1 a_2 - b_1 b_2) + (a_1 b_2 + b_1 a_2) i\text{.}\))

Answer.

Division is never a binary operation. The others are binary operations on \(\C\text{,}\) \(\R, \Q\text{,}\) and \(\Z\text{.}\) The only binary operation on \(\N\) is addition.

Activity 2.1.6.

Choose your favorite nonempty set \(X\) and describe a binary operation different than those in Investigation 2.1.5.

Answer.

Answers vary, but one option is to define \(\min : \Z\times \Z \to \Z\) by

\begin{equation*} \min(a,b) := \begin{cases} a \amp \text{if } a \le b\\ b \amp \text{if } b\le a\end{cases.} \end{equation*}

The hallmark of modern pure mathematics is the use of axioms. An axiom is essentially an unproved assertion of truth. Our use of axioms serves several purposes.

From a logical perspective, axioms help us avoid the problem of infinite regression (e.g., asking How do you know? over and over again). That is, axioms give us very clear starting points from which to make our deductions.

To that end, our first abstract algebraic structure captures and axiomatizes familiar behavior about how numbers can be combined to produce other numbers of the same type.

Definition 2.1.7.

A field is a nonempty set \(F\) with at least two elements and binary operations \(+\) and \(\cdot\text{,}\) denoted \((F,+,\cdot)\text{,}\) and satisfying the following field axioms:

  1. Given any \(a,b,c\in F\text{,}\) \((a+b)+c = a+(b+c)\text{.}\) (Associativity of addition)

  2. Given any \(a,b\in F\text{,}\) \(a+b= b+a\text{.}\) (Commutativity of addition)

  3. There exists an element \(0_F\in F\) such that for all \(a\in F\text{,}\) \(a+0_F = 0_F + a = a\text{.}\) (Additive identity)

  4. Given any \(a\in F\) there exists a \(b\in F\) such that \(a+b = b + a =0_F\text{.}\) (Additive inverse)

  5. Given any \(a,b,c\in F\text{,}\) \((a\cdot b)\cdot c = a\cdot (b\cdot c)\text{.}\) (Associativity of multiplication)

  6. Given any \(a,b\in F\text{,}\) \(a\cdot b = b\cdot a\text{.}\) (Commutativity of multiplication)

  7. There exists an element \(1_F\in F\) such that for all \(a\in F\text{,}\) \(1_F\cdot a = a\cdot 1_F = a\text{.}\) (Multiplicative identity)

  8. For all \(a\in F\text{,}\) \(a\ne 0_F\text{,}\) there exists a \(b\in F\) such that \(a\cdot b = b\cdot a = 1_F\text{.}\) (Multiplicative inverse)

  9. For all \(a,b,c\in F\text{,}\) \(a\cdot (b+c) = a\cdot b + a\cdot c\text{.}\) (Distributive property I)

  10. For all \(a,b,c\in F\text{,}\) \((a+b)\cdot c = a\cdot c + b\cdot c\text{.}\) (Distributive property II)

We will usually write \(a\cdot b\) as \(ab\text{.}\) Additionally, we will usually drop the subscripts on \(0,1\) unless we need to distinguish between fundamentally different identities in different fields.

Investigation 2.1.8.

Which of the following are fields under the specified operations? For most, a short justification or counterexample is sufficient.

  1. \(\N\) under the usual addition and multiplication operations

  2. \(\Z\) under the usual addition and multiplication operations

  3. \(2\Z\text{,}\) the set of even integers, under the usual addition and multiplication operations

  4. \(\Q\) under the usual addition and multiplication operations

  5. \(\Z_{6}\) under addition and multiplication modulo 6

  6. \(\Z_{5}\) under addition and multiplication modulo 5

  7. \(\R\) under the usual addition and multiplication operations

  8. \(\C\) under the complex addition and multiplication defined in Investigation 2.1.5

  9. \(\mathcal{M}_2(\R) := \setof{\left(\begin{matrix}a \amp b \\ c \amp d \end{matrix} \right)}{a,b,c,d\in\R}\) 1 , the set of \(2\times 2\) matrices with real coefficients using the usual definition of matrix multiplication 2  and matrix addition.

Solution.

  1. \(\N\) is not closed under taking additive inverses, so is not a field.

  2. \(\Z\) is not a field, as there is no integer \(a\) such that \(2a = 1\text{.}\)

  3. \(2\Z\) is not a field for the same reason.

  4. \(\Q\) is a field.

  5. \(\Z_{6}\) is not a field; there is no \(\overline{a}\in \Z_{10}\) for which \(\overline{2}\cdot \overline{a} = \overline{1}\) (you can check them all; there are only 4 viable options).

  6. \(\Z_{5}\) is a field. The axioms pertaining to addition and multiplication were established earlier or rely on the same axioms holding for \(\Z\text{.}\)

  7. \(\R\) is a field

  8. \(\C\) is a field. Given \(a+bi\ne 0\text{,}\) \((a+bi)^{-1} = \frac{a}{a^2 + b^2} - \frac{b}{a^2 + b^2} i\text{.}\)

  9. \(\mathcal{M}_2(\R)\) is not a field. Matrix multiplication is not commutative.

In the Investigation 2.1.8, you determined which of sets of familiar mathematical objects are and are not fields. Notice that you have been working with fields for years and that our abstraction of language to that of fields is simply to allow us to explore the common features at the same time - it is inefficient to prove the same statement about every single field when we can prove it once and for all about fields in general.

Theorem 2.1.9. Properties of Fields.

Let \(F\) be a field.

  1. The additive identity \(0\) is unique.

  2. For all \(a\in F\text{,}\) \(a \cdot 0 = 0\cdot a = 0\text{.}\)

  3. Additive inverses are unique.

  4. The multiplicative identity \(1\) is unique.

  5. Multiplicative inverses are unique.

  6. \((-1)\cdot (-1) = 1\) Hint.

    Note that we are saying that the additive inverse of the multiplicative identity times itself equals the multiplicative identity. You should use only the field axioms and the properties previously established in this theorem.

Proof.

  1. Assume \(0, 0'\) both satisfy the additive identity axiom. Observe that \(0 = 0 + 0' = 0'\text{.}\)

  2. As \(0 = 0 + 0\text{,}\) we may write \(a\cdot 0 = a\cdot (0 + 0) = a\cdot 0 + a \cdot 0\text{.}\) Now add the additive inverse of \(a\cdot 0\) to both sides to obtain \(0 = a \cdot 0 = 0\cdot a\text{.}\)

  3. Let \(a\in F\) and suppose \(b\) and \(c\) are such that \(a+c = 0\) and \(a+b = 0\text{.}\) Then \(a+c = a+b\text{,}\) and we may add \(b\) (or \(c\)) to both sides to obtain \(b+(a+b) = (b+a)+b = 0 + b = b\) and \(c+(a+c) = (c+a)+c = 0 + c = c\text{.}\) Thus \(b=c\text{.}\)

  4. Suppose \(1\) and \(1'\) are multiplicative identities. Then \(1 = 1 \cdot 1' = 1'\text{.}\)

  5. Let \(a\in F\) be nonzero and suppose \(b,c\) are multiplicative inverses for \(a\text{.}\) Then \(ab = 1 = ac\text{,}\) and we may multiply by \(b\) (or \(c\)) to obtain \(b(ab) = (ba)b = 1\cdot b = b = b(ac) = (ba)c = 1\cdot c = c\text{.}\)

  6. Observe that \((-1) + 1 = 0\) by the definition of additive inverses. Multiply both sides by \(-1\) and distribute to obtain \((-1)(-1) + (-1)\cdot 1 = (-1)\cdot 0\text{.}\) By part 2 of this theorem and the definition of a multiplicative identity, we obtain \((-1)(-1) + (-1) = 0\text{.}\) We now add the multiplicative identity to obtain \((-1)(-1) + (-1) + 1 = 0 + 1\text{,}\) which simplifies to \((-1)(-1) = 1\text{.}\)

One consequence of Theorem 2.1.9 is that, given \(a\in F\text{,}\) \(b\in F\setminus \set{0}\text{,}\) we may refer to \(-a\) as the additive inverse of \(a\text{,}\) and \(b^{-1}\) as the multiplicative inverse of \(b\text{.}\) We will thus employ this familiar terminology henceforth.

For which \(n > 1\) is \(\Z_n\) a field? Compute some examples, form a conjecture, and prove your conjecture.

Definition 4.3.1.

Let \(R\) and \(S\) be commutative rings with identity. A function \(\p : R\to S\) is a called ring homomorphism if it preserves addition and multiplication. That is, for all \(x,y\in R\text{,}\) \(\p(x+y) = \p(x) + \p(y)\) and \(\p(xy) = \p(x)\p(y)\text{.}\) If \(\p\) is a bijection, we say that \(\p\) is an isomorphism and write \(R\cong S\text{.}\) If \(\p : R\to R\) is an isomorphism, we say \(\p\) is an automorphism of \(R\text{.}\)

Checkpoint 4.3.2.

Show that the function \(\p : Z\to \Z_{5}\) defined by \(\p(x) = \overline{x}\) is a homomorphism. Is it an isomorphism? Why or why not?

Solution

Note that \(\p(x+y) = \overline{x+y} = \overline{x} + \overline{y} = \p(x) + \p(y)\text{,}\) and similarly for multiplication. We observe that \(\p\) is not an isomorphism, as \(\p(2) = \p(7) = \overline{2}\text{.}\)

Theorem 4.3.3.

Let \(R\) be commutative with identity and \(I\) an ideal of \(R\text{.}\) Define \(\p: R\to R/I\) by \(\p(r) = r+I\text{.}\) Then \(\p\) is a homomorphism.

Checkpoint 4.3.4.

Let \(R = \Z_2[x]\) and consider the map \(F: R \to R\) given by \(F(p) = p^2\text{.}\) Is \(F\) a homomorphism? An isomorphism?

Checkpoint 4.3.5.

Give an example of other rings \(R\) and \(S\) and a homomorphism \(\p : R\to S\text{.}\) Show that your function is a homomorphism and determine whether or not it is an isomorphism. Multiple presentations (of different homomorphisms) are possible.

Solution

One is complex conjugation, which is an isomorphism \(\C\to\C\text{.}\) Another is \(\p : \Z_5 \to \Z_{10}\) given by \(\p(x) = 5x\) (this isn't easy).

Definition 4.3.6.

Let \(\p : R \to S\) be a ring homomorphism. Then \(\ker \p =\setof{r\in R}{\p(r)=0_S}\) is the kernel of \(\p\text{.}\)

Checkpoint 4.3.7.

Let \(R\) be commutative with identity and \(r\in R\text{.}\) The \(r\)-evaluation homomorphism is the map \(\p_r : R[x] \to R\) defined by \(\p_r(p(x)) = p(r)\text{.}\) Prove that \(\p_r\) is a homomorphism. What is \(\ker \p_r\text{?}\)

Solution

This follows from the way sums and products of polynomials are evaluated. By definition, \(\ker \p_r\) is the set of all \(p(x)\in R[x]\) such that \(p(r) =0\text{,}\) that is, the set of all polynomials which have \(r\) as a zero.

Checkpoint 4.3.8.

Find the kernels of other homomorphisms we've looked at so far.

Theorem 4.3.9.

Given a ring homomorphism \(\p : R\to S\text{,}\) \(\ker\p\) is an ideal.

Proof.

If \(x,y\in \ker\p\text{,}\) \(\p(x+y) = \p(x) + \p(y) = 0_S + 0_S = 0_S\text{.}\) Similarly, if \(x\in \ker \p\) and \(r\in R\text{,}\) \(\p(rx) = \p(r) \p(x) = \p(r) 0_S = 0_S\text{.}\)

Theorem 4.3.10.

Let \(\p : R\to S\) be a homomorphism. Then \(\p\) is one-to-one if and only if \(\ker\p = \set{0}\text{.}\)

In what follows, we work toward a proof of the First Isomorphism Theorem for Rings.

Throughout, let \(R\) and \(S\) be commutative rings with identity, and let \(\p : R\to S\) be a homomorphism. Recall that \(\im \p = \setof{s\in S}{\p(r) = s\text{ for some } r\in R}\text{.}\)

Define \(f: R/\ker \p \to \im \p\) by \(f(r+\ker \p) = \p(r)\text{.}\)

Lemma 4.3.11.

Using the notation from above, \(f\) is a well-defined function.

Proof.

Suppose that \(r_1 + \ker\p = r_2 + \ker\p\text{.}\) Then \(r_2-r_1 \in \ker\p\text{,}\) so \(\p(r_2-r_1) = 0_S\text{,}\) and thus \(\p(r_1) = \p(r_2)\text{.}\) Therefore, \(f(r_1+\ker\p) = f(r_2+\ker\p)\text{,}\) and \(f\) is well-defined.

Lemma 4.3.12.

Using the notation above, \(f\) is a homomorphism.

Proof.

We show that \(f\) preserves addition. That it preserves multiplication will follow similarly. Observe that \(f((x+\ker\p) + (y+\ker\p)) = f((x+y)+\ker\p) = \p(x+y) = \p(x)+\p(y) = f(x+\ker\p) + f(y+\ker\p)\text{.}\)

Lemma 4.3.13.

Using the notation above, \(f\) is one-to-one.

Proof.

Suppose that \(f(r_1 + \ker\p) = f(r_2 + \ker\p)\text{.}\) That is, \(\p(r_1) = \p(r_2)\text{.}\) Then \(\p(r_1 - r_2) = 0_S\text{,}\) so \(r_1 -r_2 \in \ker\p\text{,}\) and therefore \(r_1 + \ker\p = r_2 + \ker \p\text{.}\) Thus, \(f\) is one-to-one.

Lemma 4.3.14.

Using the notation above, \(f\) is onto.

Proof.

Since \(\p\) is onto \(\im \p\) by definition, given any \(s\in \im \p\) there is some \(r\in R\) such that \(\p(r) = s\text{.}\) Then \(f(r+\ker\p) = \p(r)\text{.}\)

We thus obtain:

Theorem 4.3.15. First Isomorphism Theorem.

Let \(\p : R\to S\) be a homomorphism of commutative rings. Then \(R/\ker \p \cong \im \p\text{.}\)

In particular, if \(\p : R\to S\) is onto, \(R/\ker \p \cong S\text{.}\)

Theorem 4.3.16.

We have the following isomorphisms of rings.

  1. \(\Z/\ideal{m} \cong \Z_m\)

  2. \(\Q[x]/\ideal{x-5} \cong \Q\)

  3. \(\R[x]/\ideal{x^2+1} \cong \C\)

Proof.

Define \(\p: \R[x] \to \C\) by \(\p(p(x)) = p(i)\text{.}\) We saw earlier that this evaluation map is a homomorphism. It is easy to see that \(\p\) is onto as \(\p(a+bx) = a+bi\) for any \(a,b\in \R\text{.}\) Thus, \(\R[x]/\ker\p \cong \C\text{.}\)

We claim that \(\ker\p = \ideal{x^2+1}\text{.}\) Clearly, \(\ideal{x^2+1} \subseteq \ker\p\text{.}\) Moreover, \(\ker\p \subsetneq \R[x]\text{.}\) Finally, \(\ideal{x^2+1}\) is prime and thus maximal, as \(\R[x]\) is a PID. Thus, \(\ker\p = \ideal{x^2+1}\text{.}\)

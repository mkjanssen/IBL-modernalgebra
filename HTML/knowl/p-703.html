Following the hint and LemmaÂ 4.1.11, we have \(I = \ideal{x_1, x_2, \ldots, x_n}\) while \(I_j\subseteq I\) for all \(j\text{.}\) Since \(I = \bigcup\limits_{j=1}^\infty I_j\text{,}\) for each \(k, 1\le k \le n\text{,}\) there is a \(j_k\) satisfying \(x_k \in I_{j_k}\text{.}\) What is more, we may rename \(x_1, x_2, \ldots, x_n\) so that \(j_1 \le j_2 \le \ldots \le j_n\) and thus

\begin{equation*} I_{j_1} \subseteq I_{j_2}\subseteq \cdots \subseteq I_{j_n}. \end{equation*}

Observe that this means that \(x_{j_k}\in I_{j_n}\) for all \(k\text{,}\) so that \(I = I_{j_n}\text{.}\) Thus, the chain stabilizes, and \(R\) is Noetherian.

Suppose that \(I = \ideal{a(x)}\text{.}\) Since \(2\in I\text{,}\) \(2 = c(x) a(x)\) for some \(c(x)\in \Z[x]\text{,}\) so \(\deg c(x) = \deg a(x) = 0\text{,}\) and thus \(a(x) = a\in \Z\text{.}\) We also have \(x\in I\text{,}\) so \(x = b(x) a\text{,}\) which implies that \(\deg b(x) = 1\text{,}\) so \(b(x) = b_1 x + b_0\) for some \(b_1,b_0\in \Z\text{.}\) Then \(x = (b_1 x + b_0)a\) implies \(b_1 a = 1\text{.}\) Thus, \(a = 1\) or \(a = -1\text{.}\) In either case, \(a\) is a unit in \(\Z[x]\) so \(1\in I\text{.}\) By TheoremÂ 4.1.7, there are polynomials \(p(x),q(x)\in\Z[x]\) such that \(1 = 2p(x) + xq(x)\text{.}\) If \(p_0\) is the constant term of \(p(x)\text{,}\) it follows that the constant term of \(2p(x) + xq(x)\) is \(2p_0 = 1\text{,}\) which is a contradiction.

Thus, \(I\) is not principal.

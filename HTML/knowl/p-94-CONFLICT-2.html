Let \(p\) be prime and suppose \(m\in \N\) divides \(p\text{,}\) so \(p = mk\) for some \(k\in \N\text{.}\) By definition, \(p|m\) or \(p|k\text{.}\) If \(p|m\text{,}\) then \(pj = m\text{,}\) so \(p = pjk\text{,}\) and \(jk=1\text{,}\) which means \(j=k=1\) and thus \(m=p\text{.}\) If \(p|k\text{,}\) then \(pj=k\text{,}\) so \(p = mpj\text{,}\) and \(mj=1\text{,}\) so \(m = j = 1\text{.}\)

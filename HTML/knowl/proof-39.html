Proof

Existence: If \(f(x) = 0\text{,}\) then \(q(x) = 0\) and \(r(x) = 0\) will do. If \(f(x)\ne 0\) and \(\deg f \lt \deg g\text{,}\) then \(q(x) = 0\) and \(r(x) = f(x)\) will suffice. Thus, we need only consider the case in which \(f(x) \ne 0\) and \(\deg f \ge \deg g\text{.}\) We use induction on \(\deg f = m\text{.}\)

When \(m = 0\text{,}\) \(\deg g \le 0 = \deg f\text{,}\) and as \(g(x) \ne 0\text{,}\) both \(f\) and \(g\) are nonzero constants. Then \(r = 0\) and \(q = fg^{-1}\) will work.

Now assume \(q\) and \(r\) exist whenever \(\deg f \lt m\text{.}\) Assume \(\deg f = m\) and write \(f(x) = a_0 + a_1 x + \cdots + a_m x^m\) and \(g(x) = b_0 + b_1 x + \cdots + b_n x^n\text{.}\) Use LemmaÂ 2.4.12 and set \(h(x) = f(x) - a_m b_n^{-1} x^{m-n} g(x)\text{,}\) which must have degree less than \(f\text{.}\) Thus, by induction, there exist \(q_1, r_1\in F[x]\) such that \(h = g q_1 + r_1\text{,}\) with \(r_1 = 0\) or \(\deg r_1 \lt \deg g\text{.}\)

We therefore have

\begin{align*} f(x) \amp = a_m b_n^{-1} x^{m-n} g(x) + h(x)\\ \amp = a_m b_n^{-1} x^{m-n} g(x) + g(x) q_1(x) + r_1(x)\\ \amp = (a_m b_n^{-1} x^{m-n} + q_1(x)) g(x) + r_1(x)\text{,} \end{align*}

where \(q = a_m b_n^{-1} x^{m-n} + q_1(x)\) and \(r = r_1(x)\) have the desired properties.

Uniqueness: Suppose \(f = gq + r\) and \(f = g\hat{q} + \hat{r}\text{,}\) where \(r,\hat{r}\) both have the desired properties. Then

\begin{equation*} 0 = g[q - \hat{q}] + [r - \hat{r}]\text{,} \end{equation*}

or \(\hat{r} - r = g [q - \hat{q}]\text{.}\) Thus either \(\hat{r} - r = 0\text{,}\) or \(\hat{r} - r\) has degree at least \(\deg g\text{.}\) The latter is clearly impossible, so \(\hat{r} = r\) and \(\hat{q} = q\text{.}\)

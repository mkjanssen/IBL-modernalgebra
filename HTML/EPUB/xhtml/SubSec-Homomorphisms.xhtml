Section 4.2 Homomorphisms

Guiding Questions.

In this section, we'll seek to answer the questions:

  • What are subrings, and how can we tell if a given subset of a ring is a subring?

Definition 4.2.1 .

Let \(R\) and \(S\) be commutative rings with identity. A function \(\p : R\to S\) is a called ring homomorphism if it preserves addition, multiplication, and sends the identity of \(R\) to the identity of \(S\text{.}\) That is, for all \(x,y\in R\text{:}\)

  • \(\p(x+y) = \p(x) + \p(y)\text{,}\)
  • \(\p(xy) = \p(x)\p(y)\text{,}\) and
  • \(\p(1_R) = 1_S\text{.}\)

If \(\p\) is a bijection, we say that \(\p\) is an isomorphism and write \(R\cong S\text{.}\) If \(\p : R\to R\) is an isomorphism, we say \(\p\) is an automorphism of \(R\text{.}\)

Activity 4.2.1 .

Show that the function \(\p : Z\to \Z_{5}\) defined by \(\p(x) = \overline{x}\) is a homomorphism. Is it an isomorphism? Why or why not?

Solution

Note that \(\p(x+y) = \overline{x+y} = \overline{x} + \overline{y} = \p(x) + \p(y)\text{,}\) and similarly for multiplication. We observe that \(\p\) is not an isomorphism, as \(\p(2) = \p(7) = \overline{2}\text{.}\)

Investigation 4.2.2 .

Let \(R = \Z_2[x]\) and consider the map \(F: R \to R\) given by \(F(p) = p^2\text{.}\) Is \(F\) a homomorphism? An isomorphism?

Exploration 4.2.3 .

Give an example of other rings \(R\) and \(S\) and a homomorphism \(\p : R\to S\text{.}\) Show that your function is a homomorphism and determine whether or not it is an isomorphism. Multiple presentations (of different homomorphisms) are possible.

Solution

One is complex conjugation, which is an isomorphism \(\C\to\C\text{.}\) Another is \(\p : \Z_5 \to \Z_{10}\) given by \(\p(x) = 5x\) (this isn't easy).

Definition 4.2.2 .

Let \(\p : R \to S\) be a ring homomorphism. Then \(\ker \p =\setof{r\in R}{\p(r)=0_S}\) is the kernel of \(\p\text{.}\)

Theorem 4.2.3 .

Let \(R\) be commutative with identity and \(r\in R\text{.}\) The \(r\)-evaluation map \(\p_r : R[x] \to R\) defined by \(\p_r(p(x)) = p(r)\) is a homomorphism.

Solution

This follows from the way sums and products of polynomials are evaluated.

Exploration 4.2.4 .

What is \(\ker \p_r\text{?}\)

SolutionBy definition, \(\ker \p_r\) is the set of all \(p(x)\in R[x]\) such that \(p(r) =0\text{,}\) that is, the set of all polynomials which have \(r\) as a zero.
Activity 4.2.5 .

Find the kernels of other homomorphisms we've looked at so far.

SolutionAnswers will vary.
Theorem 4.2.4 .

Given a ring homomorphism \(\p : R\to S\text{,}\) \(\ker\p\) is an ideal.

Proof.

If \(x,y\in \ker\p\text{,}\) \(\p(x+y) = \p(x) + \p(y) = 0_S + 0_S = 0_S\text{.}\) Similarly, if \(x\in \ker \p\) and \(r\in R\text{,}\) \(\p(rx) = \p(r) \p(x) = \p(r) 0_S = 0_S\text{.}\)

Theorem 4.2.5 .

Let \(\p : R\to S\) be a homomorphism. Then \(\p\) is one-to-one if and only if \(\ker\p = \set{0}\text{.}\)

Section 1.2 Divisibility and GCDs in the Integers

Guiding Questions.

In this section, we'll seek to answer the questions:

  • What does it mean for one integer to divide another?

  • What properties does divisibility enjoy in the integers?

  • What is the greatest common divisor of two integers?

  • How can we compute the greatest common divisor of two integers?

Subsection 1.2.1 Divisibility and the Division Algorithm

In this section, we begin to explore some of the arithmetic and algebraic properties of \(\Z\text{.}\) We focus specifically on the divisibility and factorization properties of the integers, as these are the main focus of the text as a whole. One of the primary goals of this section is to formalize definitions that you are likely already familiar with and of which you have an intuitive understanding. At first, this might seem to unnecessarily complicate matters. However, it will become clear as we move forward that formal mathematical language and notation are necessary to extend these properties to a more abstract setting. We begin with a familiar notion.

Definition 1.2.1.

Let \(a,b\in \Z\text{.}\) We say that \(a\) divides \(b\text{,}\) and write \(a\mid b\text{,}\) if there is an integer \(c\) such that \(ac = b\text{.}\) In this case, say that \(a\) and \(c\) are factors of \(b\text{.}\) If no such \(c\in \Z\) exists, we write \(a\nmid b\text{.}\)

Note that the symbol \(|\) is a verb; it is therefore correct to say, e.g., \(2|4\text{,}\) as 2 does divide 4. However, it is an abuse of notation to say that \(2\mid 4 = 2\text{.}\) Instead, we likely mean \(4\div 2 = 2\) or \(\frac{4}{2} = 2\) (though we will not deal in fractions just yet).

We next collect several standard results about divisibility in \(\Z\) which will be used extensively in the remainder of this text.

Theorem 1.2.3.

Let \(a,b,c\in\Z\text{.}\) If \(a\mid b\) and \(a\mid c\text{,}\) then \(a\mid (b+c)\text{.}\)

Proof.

Our hypothesis means that there exist integers \(k_1\) and \(k_2\) such that \(b = a k_1\) and \(c = a k_2\text{.}\) Then \(b+c = a k_1 + a k_2 = a(k_1 + k_2)\text{.}\) Let \(k' = k_1 + k_2\text{,}\) and observe that \(k'\) is an integer; then \(b+c = a k'\text{,}\) so \(a\mid (b+c)\text{.}\)

Theorem 1.2.4.

Let \(a,b,c\in\Z\text{.}\) If \(a\mid b\text{,}\) then \(a\mid bc\text{.}\)

Proof.

Our hypotheses means there is an integer \(k\) for which \(b = ak\text{.}\) Then \(bc = (ak)c = a(kc)\text{.}\) We observe that \(k' = kc\) is an integer, so \(bc = ak'\text{,}\) and therefore \(a\mid bc\text{.}\)

Investigation 1.2.5.

Consider the following partial converse to Theorem 1.2.4: If \(a,b,c\in\Z\) with \(a|bc\text{,}\) must \(a|b\) or \(a|c\text{?}\) Supply a proof or give a counterexample.

Solution.

The converse is false: \(6| 3\cdot 4\text{,}\) but \(6\nmid 3\) and \(6\nmid 4\text{.}\)

Theorem 1.2.6.

Let \(a,b,c,d\in \Z\text{.}\) If \(a = b+c\) and \(d\) divides any two of \(a,b,c\text{,}\) then \(d\) divides the third.

Proof.

We consider two cases.

Case 1: Assume without loss of generality that \(d|a\) and \(d|b\text{.}\) Then there are integers \(k_1, k_2\) such that \(d k_1 = a\) and \(d k_2 = b\text{.}\) We observe that \(c = a - b = d k_1 - d k_2 = d(k_1 - k_2)\text{,}\) so \(d|c\text{.}\)

Case 2: Assume that \(d| b\) and \(d|c\text{,}\) so there are \(k_1,k_2\in \Z\) such that \(d k_1 = b\) and \(d k_2 = c\text{.}\) Then \(a = b+ c = dk_1 + dk_2 = d(k_1 + k_2)\text{,}\) so \(d|a\text{.}\)

Investigation 1.2.7.

Formulate a conjecture akin to the previous theorems about divisibility in \(\Z\text{,}\) and then prove it.

Solution.

Answers vary.

As we saw above, not all pairs of integers \(a,b\) satisfy \(a\mid b\) or \(b\mid a\text{.}\) However, our experience in elementary mathematics does apply: there is often something left over (a remainder) 1 . The following theorem formalizes this idea for \(a,b\in \N\text{.}\)

Theorem 1.2.8. The Division Algorithm for \(\N\).

Let \(a,b\in \N\text{.}\) Then there exist unique integers \(q,r\) such that \(a = bq + r\text{,}\) where \(0 \le r \lt b\text{.}\)

Hint 1.

There are two parts to this theorem. First, you must establish that \(q\) and \(r\) exist. This is best done via Axiom 1.1.2. If you're stuck on that, check the second hint.

Once you have established that \(q\) and \(r\) exist, show that they are unique but assuming \(a = bq+r\) and \(a = bq' + r'\text{,}\) where \(r,r'\) both satisfy the conditions of the theorem. Argue that \(q = q'\) and \(r = r'\text{.}\)

Hint 2.

Let \(S = \setof{a-bs}{s\in \N_0, a-bs\ge 0}\text{.}\)

Proof.

First, assume that \(a \lt b\text{.}\) Then \(q = 0\) and \(r = a\) are sufficient. If \(a = b\text{,}\) then \(q = 1\) and \(r = 0\) will work.

Now, assume that \(a > b\text{.}\) Then the set \(S = \setof{s\in\N}{a-bs \ge 0}\) is nonempty, as \(a-b\in S\text{.}\) By Axiom 1.1.2, \(S\) has a least element, which we will call \(r\text{.}\) Let \(q\in \N_0\) be such that \(r = a-bq\text{;}\) then \(a = bq + r\text{.}\)

To finish the proof, it is enough to show that this choice of \(r\) satisfies \(0\le r \le b-1\text{.}\) Observe that \(r\in S\text{,}\) so \(r\ge 0\text{.}\) If \(r \ge b\text{,}\) then \(a - b(q+1) = a-bq -b = r -b \ge 0\text{.}\) Since \(q+1 > q\text{,}\) this is a contradiction to the assumption that \(q\) was the largest element of \(S\text{.}\) Thus, \(r \lt b\text{,}\) i.e., \(r \le b-1\text{.}\)

Let \(q,q',r,r'\) be such that \(a = bq+r\) and \(a=bq'+r'\text{.}\) We see that \(bq+r = bq' + r'\text{,}\) so \(b(q-q') = r'-r\text{,}\) so \(b|r'-r\text{.}\) If \(r'-r = 0\text{,}\) we are done.

Otherwise, \(r'-r\) is a nonzero multiple of \(n\text{.}\) If \(r' \ge r\text{,}\) then \(0\le b(q-q') = r' - r \le b-1 - r\text{,}\) a contradiction. Similarly, if \(r' \le r\text{,}\) write \(b(q'-q) = r-r' \ge 0\text{,}\) and a symmetric argument leads to a contradiction.

Thus, \(r' = r\text{,}\) so \(b(q-q') = 0\text{,}\) and since \(b > 0\text{,}\) \(q - q'= 0\text{,}\) i.e., \(q = q'\text{.}\)

Warning!

This theorem has two parts: existence and uniqueness. Do not try to prove them both at the same time.

Unsurprisingly, the Division Algorithm also holds in \(\Z\text{,}\) though the existence of negative integers requires a careful restatement.

Subsection 1.2.2 Greatest Common Divisors

We next turn to another familiar property of the integers: the existence of greatest common divisors.

Definition 1.2.10.

Let \(a,b\in \Z\) such that \(a\) and \(b\) are not both 0. A greatest common divisor of \(a\) and \(b\text{,}\) denoted \(\gcd(a,b)\text{,}\) is a natural number \(d\) satisfying

  1. \(d\mid a\) and \(d\mid b\)

  2. if \(e\in \N\) and \(e\mid a\) and \(e\mid b\text{,}\) then \(e\mid d\text{.}\)

If \(\gcd(a,b) = 1\text{,}\) we say that \(a\) and \(b\) are relatively prime or coprime.

This definition may be different than the one you are used to, which likely stated that \(d \ge e\) rather than condition 2 in Definition 1.2.10. It can be proved using the order relations of \(\Z\) that the definition given here is equivalent to that one. However, we will prefer this definition, as it generalizes naturally to other number systems which do not have an order relation like \(\Z\text{.}\)

Most such methods rely in some form on our ability to factor integers. However, the problem of factoring arbitrary integers is actually surprisingly computationally intensive. Thankfully, there is another way to compute \(\gcd(a,b)\text{,}\) to which we now turn.

Theorem 1.2.12.

Let \(a,b,c\in\Z\) such that \(a = b+c\) with \(a\) and \(b\) not both zero. Then \(\gcd(a,b) = \gcd(b,c)\text{.}\)

Proof.

Let \(d = \gcd(a,b)\) and \(e = \gcd(b,c)\text{.}\) Since \(d|a\) and \(d|b\text{,}\) \(d|c\text{,}\) and so \(d|e\) by definition.

Similarly, since \(e|b\) and \(e|c\text{,}\) \(e|a\text{,}\) so \(e|d\) by definition.

Thus, \(e = d\text{.}\)

Investigation 1.2.14. (Euclidean Algorithm).

Let \(a,b\in \N\text{.}\) Use Theorem 1.2.8 and Investigation 1.2.13 to determine an algorithm for computing \(\gcd(a,b)\text{.}\) How could your method be modified to compute \(\gcd(a,b)\) for \(a,b\in\Z\text{?}\)

Solution.

Let \(a,b\in \N\text{.}\) Write \(a = bq_1+r_1\text{.}\) By Theorem 1.2.13, \(\gcd(a,b) = \gcd(b,r_1)\text{.}\)

Then, write \(b = r_1 q_2 + r_2\) using the Division algorithm. Observe \(\gcd(b,r_1) = \gcd(r_1,r_2)\text{.}\) Continue until there is a \(k\) for which \(r_k = 0\text{,}\) so \(\gcd(r_k,r_{k-1}) = r_{k-1}\text{.}\) Then \(r_{k-1}= \gcd(r_k, r_{k-1}) = \gcd(r_{k-1},r_{k-2}) = \cdots = \gcd(a,b)\text{.}\)

Activity 1.2.15.

Use the Euclidean algorithm to compute \(\gcd(18489,17304)\text{.}\)

Solution.

We write

\begin{align*} 18489 \amp = 17304 \cdot 1 +1185\\ 17304 \amp = 1185\cdot 14 +714\\ 1185 \amp = 714 \cdot 1 + 471\\ 714 \amp = 471 \cdot 1+ 243\\ 471 \amp = 243\cdot 1 + 228\\ 243 \amp = 228 \cdot 1+ 15\\ 228 \amp = 15 \cdot 15 + 3\\ 15 \amp = 3\cdot 5 + 0\text{.} \end{align*}

So \(\gcd(18489,17304)=3\text{.}\)

The following identity provides a useful characterization of the greatest common divisor of two integers, not both zero. We will return to this idea several times, even after we have left the familiar realm of the integers.

Theorem 1.2.16. Bézout's Identity.

For any integers \(a\) and \(b\) not both 0, there are integers \(x\) and \(y\) such that

\begin{equation*} ax + by = \gcd(a,b)\text{.} \end{equation*}
Hint 1.

Apply Axiom 1.1.2 to a well-chosen set.

Hint 2.

Apply Axiom 1.1.2 to \(S = \setof{as+bt}{s,t\in\Z, \ as+bt \gt 0}\text{.}\)

Proof.

Consider the set \(S = \setof{as+bt}{s,t\in\Z, \ as+bt \gt 0}\text{.}\) We first show that \(S\ne \emptyset\text{.}\)

Suppose without loss of generality that \(a\ne 0\text{,}\) and consider \(a \gt 0\text{;}\) then \(s=1, t = 0\) is sufficient to guarantee \(as+bt\in S\text{.}\) If \(a \lt 0\text{,}\) then \(s = -1, t= 0\) is sufficient. Regardless, \(S\ne \emptyset\text{,}\) and by the Well-Ordering Principle, \(S\) has a least element, \(d = ax+by\text{.}\) We claim \(d = \gcd(a,b)\text{.}\)

We first show that \(d|a\) Use the division algorithm to write \(a = dq + r\text{,}\) where \(r \lt d\text{.}\) We find \(r = a - dq = a - (ax+by)q = a(1-xq) + by\text{,}\) and if \(r\gt 0\text{,}\) \(r\in S\text{,}\) contradicting the status of \(d\) as the least element of \(S\text{.}\) Thus, \(r = 0\text{,}\) and \(d|a\text{.}\) By a symmetric proof, \(d|b\text{.}\)

Suppose then that \(e\) is a common divisor of \(a\) and \(b\text{;}\) write \(a = e k_1\) and \(b = e k_2\text{.}\) Then \(d = ax + by = (ek_1)x + (ek_2)y = e(k_1x + k_2 y)\text{,}\) whence \(e|d\text{.}\) Therefore, \(d = \gcd(a,b)\text{.}\)

We conclude with an answer to the questions raised by Investigation 1.2.5.

Theorem 1.2.17.

Let \(a, b\text{,}\) and \(c\) be integers. If \(a|bc\) and \(\gcd(a,b) = 1\text{,}\) then \(a|c\text{.}\)

Proof.

Suppose \(\gcd(a,b) = 1\text{.}\) By Theorem 1.2.16, there are integers \(x\) and \(y\) such that

\begin{equation*} ax+by = 1\text{.} \end{equation*}

Multiply by \(c\) to get

\begin{equation*} acx + bcy = c\text{.} \end{equation*}

Since \(a|bc\) by assumption, there is an integer \(k\) such that \(ak = bc\text{,}\) so we have

\begin{equation*} acx + bcy = acx + aky = a(cx+ky) = c\text{,} \end{equation*}

so \(a|c\text{.}\)

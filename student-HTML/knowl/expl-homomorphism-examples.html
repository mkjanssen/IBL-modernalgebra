Exploration 4.2.2.

Determine whether the following functions are homomorphisms, isomorphisms, automorphisms, or none of these. Note that \(R\) denotes an arbitrary commutative ring with identity and \(F\) a field.
  1. \(\p : R\to R\) defined by \(\p(x)=x\)
  2. \(\psi : R\to R\) defined by \(\psi(x)=-x\)
  3. \(\alpha : \Z\to \Z\) defined by \(\alpha(x)=5x\)
  4. \(F : \Z_2[x]\to \Z_2[x]\) defined by \(F(p) = p^2\)
  5. \(\iota : \C\to \C\) defined by \(\iota(a+bi)=a-bi\text{,}\) where \(a,b\in \R, i^2 = -1\)
  6. \(\beta : \Z\to \Z_{5}\) defined by \(\beta(x) = \overline{x}\)
  7. \(\epsilon_r : F[x] \to F\) defined by \(\epsilon_r(p(x)) = p(r)\text{,}\) where if \(p(x) = a_0 + a_1 x + \cdots + a_n x^n\text{,}\) \(p(r)\) is the expression obtained by “plugging \(r\) into \(p\)”: \(p(r) = a_0 + a_1 r + \cdots + a_n r^n\) (this is known as the \(r\)-evaluation map)
  8. \(\xi : \Z_5 \to \Z_{10}\) defined by \(\xi(\overline{x}) = \overline{5x}\)

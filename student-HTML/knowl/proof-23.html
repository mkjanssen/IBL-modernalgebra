Proof

  1. Assume \(0, 0'\) both satisfy the additive identity axiom. Observe that \(0 = 0 + 0' = 0'\text{.}\)

  2. As \(0 = 0 + 0\text{,}\) we may write \(a\cdot 0 = a\cdot (0 + 0) = a\cdot 0 + a \cdot 0\text{.}\) Now add the additive inverse of \(a\cdot 0\) to both sides to obtain \(0 = a \cdot 0 = 0\cdot a\text{.}\)

  3. Let \(a\in F\) and suppose \(b\) and \(c\) are such that \(a+c = 0\) and \(a+b = 0\text{.}\) Then \(a+c = a+b\text{,}\) and we may add \(b\) (or \(c\)) to both sides to obtain \(b+(a+b) = (b+a)+b = 0 + b = b\) and \(c+(a+c) = (c+a)+c = 0 + c = c\text{.}\) Thus \(b=c\text{.}\)

  4. Suppose \(1\) and \(1'\) are multiplicative identities. Then \(1 = 1 \cdot 1' = 1'\text{.}\)

  5. Let \(a\in F\) be nonzero and suppose \(b,c\) are multiplicative inverses for \(a\text{.}\) Then \(ab = 1 = ac\text{,}\) and we may multiply by \(b\) (or \(c\)) to obtain \(b(ab) = (ba)b = 1\cdot b = b = b(ac) = (ba)c = 1\cdot c = c\text{.}\)

  6. Observe that \((-1) + 1 = 0\) by the definition of additive inverses. Multiply both sides by \(-1\) and distribute to obtain \((-1)(-1) + (-1)\cdot 1 = (-1)\cdot 0\text{.}\) By part 2 of this theorem and the definition of a multiplicative identity, we obtain \((-1)(-1) + (-1) = 0\text{.}\) We now add the multiplicative identity to obtain \((-1)(-1) + (-1) + 1 = 0 + 1\text{,}\) which simplifies to \((-1)(-1) = 1\text{.}\)

Answer 2.4.6.1.

We have “\(z\in \ideal{x} \Leftrightarrow x|z\)” and “\(\ideal{x}\subseteq \ideal{y} \Leftrightarrow y|x\)” 1 .
Note that \(z\in \ideal{x} \Leftrightarrow \exists r\in R, z = xr \Leftrightarrow x|z\text{.}\)
Similarly, suppose \(\ideal{x}\subseteq \ideal{y}\text{.}\) Then \(x\in \ideal{y}\text{,}\) so \(y|x\text{.}\) Conversely, if \(y|x\text{,}\) then there is some \(r\in R\) such that \(x = yr\text{,}\) and thus for all \(ax\in \ideal{x}\text{,}\) \(ax = (ar)y \in \ideal{y}\text{.}\)
Note that this means that if we want to know if \(\ideal{x}\subseteq \ideal{y}\text{,}\) it's enough to check that \(x\in \ideal{y}\text{.}\)

Assume \(R\) is a domain and \(ac = bc\text{.}\) Then \(ac - bc = 0\text{,}\) so \(c(a-b) = 0\text{.}\) Since \(R\) is a domain, it has no zero divisors, and therefore either \(c = 0\) or \(a-b=0\text{.}\) The first possibility is ruled out by our assumptions on \(a,b,c\text{,}\) so we must have \(a-b = 0\text{,}\) or \(a =b\text{.}\)

Conversely, assume that \(R\) is not a domain. Let \(z\in R\) be a zero divisor; then there is a nonzero \(x\in R\) such that \(xz = 0 = 0 z\text{.}\) This implies that \(x = 0\text{,}\) a contradiction.

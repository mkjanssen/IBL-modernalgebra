Proof

Let \(R\) be a euclidean domain, and \(I\) an ideal of \(R\text{.}\) If \(I = \set{0}\text{,}\) then \(I\) is principal, so assume that \(I\ne \set{0}\text{.}\)

Define \(S = \setof{\delta(x)>0}{x\in I}\text{.}\) Then the Well-Ordering Principle guarantees that \(S\) has a least element. Let \(d\in I\) be such that \(\delta(d) > 0\) is minimal. We claim that \(I = \ideal{d}\text{.}\)

Clearly \(\ideal{d}\subseteq I\text{.}\) Now assume that \(a\in I\text{,}\) and write \(a = dq + r\text{,}\) where either \(r = 0\) or \(r\ne 0\) and \(\delta(r) \lt \delta(d)\text{.}\) If \(\delta(r) \lt \delta(d)\text{,}\) then \(r = a - dq \in I\text{,}\) contradicting the minimality of \(\delta(d)\text{.}\) Thus, \(r = 0\text{,}\) and \(a\in \ideal{d}\text{.}\)

Therefore, \(R\) is a PID.

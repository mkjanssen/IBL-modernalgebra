A field is a nonempty set \(F\) with at least two elements and binary operations \(+\) and \(\cdot\text{,}\) denoted \((F,+,\cdot)\text{,}\) and satisfying the following field axioms:
  1. Given any \(a,b,c\in F\text{,}\) \((a+b)+c = a+(b+c)\text{.}\) (Associativity of addition)
  2. Given any \(a,b\in F\text{,}\) \(a+b= b+a\text{.}\) (Commutativity of addition)
  3. There exists an element \(0_F\in F\) such that for all \(a\in F\text{,}\) \(a+0_F = 0_F + a = a\text{.}\) (Additive identity)
  4. Given any \(a\in F\) there exists a \(b\in F\) such that \(a+b = b + a =0_F\text{.}\) (Additive inverse)
  5. Given any \(a,b,c\in F\text{,}\) \((a\cdot b)\cdot c = a\cdot (b\cdot c)\text{.}\) (Associativity of multiplication)
  6. Given any \(a,b\in F\text{,}\) \(a\cdot b = b\cdot a\text{.}\) (Commutativity of multiplication)
  7. There exists an element \(1_F\in F\) such that for all \(a\in F\text{,}\) \(1_F\cdot a = a\cdot 1_F = a\text{.}\) (Multiplicative identity)
  8. For all \(a\in F\text{,}\) \(a\ne 0_F\text{,}\) there exists a \(b\in F\) such that \(a\cdot b = b\cdot a = 1_F\text{.}\) (Multiplicative inverse)
  9. For all \(a,b,c\in F\text{,}\) \(a\cdot (b+c) = a\cdot b + a\cdot c\text{.}\) (Distributive property I)
  10. For all \(a,b,c\in F\text{,}\) \((a+b)\cdot c = a\cdot c + b\cdot c\text{.}\) (Distributive property II)

Proof

Observe that \(0 = 0\cdot a \in \ideal{a}\text{.}\)
Moreover, if \(r_1 a, r_2 a\in \ideal{a}\text{,}\) \(r_1 a - r_2 a = (r_1 - r_2)a \in \ideal{a}\text{.}\) Finally, if \(x\in R\) and \(r a\in \ideal{a}\text{,}\) \(x (ra) = (xr) a \in \ideal{a}\text{.}\)
Thus, \(\ideal{a}\) is an ideal.

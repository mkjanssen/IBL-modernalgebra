Proof

Let \(p\) be prime and suppose \(m\in \N\) divides \(p\text{,}\) so \(p = mk\) for some \(k\in \N\text{.}\) By definition, \(p|m\) or \(p|k\text{.}\) If \(p|m\text{,}\) then \(pj = m\text{,}\) so \(p = pjk\text{,}\) and \(jk=1\text{,}\) which means \(j=k=1\) and thus \(m=p\text{.}\) If \(p|k\text{,}\) then \(pj=k\text{,}\) so \(p = mpj\text{,}\) and \(mj=1\text{,}\) so \(m = j = 1\text{.}\)

Conversely, assume \(p\in \N\) has the property that whenever an integer \(m\) exists with \(m|p\text{,}\) then \(m= p\) or \(m =1\text{.}\) Suppose \(p|ab\text{,}\) so there exists a \(k\in\N\) such that \(pk = ab\text{.}\) Note that if \(\gcd(a,p) = d > 1\text{,}\) then \(d|p\text{,}\) and so \(d = p\text{,}\) and then \(p|a\text{.}\) If \(\gcd(a,p) = 1\text{,}\) then by BÃ©zout's Identity there are integers \(x\) and \(y\) such that \(ax + py = 1\text{,}\) so we may multiply by \(b\) to obtain \(abx + pby = b\text{,}\) whence \(p|b\text{.}\)

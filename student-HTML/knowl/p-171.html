Since \(t_i - r_i > 0\text{,}\) we have that
\begin{equation*} p_i | p_1^{r_1} p_2^{r_2} \cdots \hat{p}_i^{r_i} \cdots p_m^{r_m}\text{,} \end{equation*}
from which \(p_i | p_j\) for some \(j\ne i\) by Lemma 2.8. This is a contradiction, so \(r_i = t_i\) for \(i = 1, 2, \ldots, m\text{.}\)

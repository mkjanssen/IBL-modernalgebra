Activity 4.3.7.

Let \(R = \Z_6\) and define \(\p : \Z_6 \to \Z_2\) by \(\p(\overline{x}) = \overline{x}\text{.}\) That is, \(\p\) sends an equivalence class \(\overline{x}\in \Z_6\) represented by \(x\in \Z\) to the equivalence class represented by \(x\) in \(\Z_2\text{.}\)

  1. Show that \(\p\) is a well-defined function.

  2. Prove that \(\p\) is a homomorphism.

  3. Is \(\p\) onto? Justify.

  4. Compute \(\ker\p\) (that is, list the elements in the set). Is \(\p\) one-to-one?

  5. Without appealing to the definition, is \(\ker\p\) prime? Maximal? Explain.

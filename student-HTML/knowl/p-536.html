Let \(g(x)\in I\) and use Theorem 2.4.14 to write \(g(x) = f(x) q(x) + r(x)\text{,}\) where \(\deg r(x) \lt \deg f(x)\) or \(r(x) = 0\text{.}\) As in Theorem 2.4.8, write \(r(x) = g(x) - f(x) q(x) \in I\text{,}\) so we must have that \(r(x) = 0\text{.}\) Thus, \(f(x) | g(x)\) and \(g(x)\in \ideal{f(x)}\text{.}\)

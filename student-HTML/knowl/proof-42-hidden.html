Observe that \(0 = x0 + y0 \in I\text{.}\) Additionally, if \(x m_1 + y n_1, x m_2 + y n_2\in I\text{,}\) then \((x m_1 + y n_1) + (x m_2 + y n_2) = x (m_1 + m_2) + y (n_1 + n_2) \in I\text{,}\) and \(r(x m_1 + y n_1) = x (rm_1) + y(rn_1) \in I\text{.}\) Thus, \(I\) is an ideal.

Since \(R\) is a PID, there exists \(d\in R\) such that \(I = \ideal{d}\text{.}\) We claim that \(d\) is a GCD of \(x\) and \(y\text{.}\)

It is clear that \(d|x\text{,}\) as \(x\cdot 1 + y\cdot 0\in I = \ideal{d}\text{.}\) Similarly, \(d|y\text{.}\)

Now let \(e\in R\) be a common divisor of \(x\) and \(y\text{.}\) We wish to show that \(e|d\text{.}\) Write \(x = e k_1\) and \(y = e k_2\text{.}\) Since \(d\in \ideal{d}\text{,}\) there exist \(s,t\in R\) such that \(d = x s + yt = (e k_1) s + (e k_2)t = e(k_1 s + k_2 t)\text{,}\) and thus \(e|d\text{.}\)

In particular, there exist \(s,t\in R\) such that a GCD \(d\) of \(x\) and \(y\) can be written as \(d = xs + yt\text{.}\)

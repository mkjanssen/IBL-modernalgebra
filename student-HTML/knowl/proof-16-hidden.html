We first state a claim that will be useful.

Claim 1: If \(p,q\) are primes such that \(p|q\text{,}\) then \(p = q\text{.}\)

Proof of Claim 1: Since \(p, q > 1\) and both \(p\) and \(q\) are prime, \(p = q\) by the definition of primality. \(\checkmark\)

Now assume that \(pk = q_1 q_2 \cdots q_n = (q_1 q_2 \cdots q_{n-1}) q_n\text{.}\) If \(p|q_n\text{,}\) we're done by Claim 1. If not, the definition of a prime guarantees that \(p| q_1 q_2 \cdots q_{n-1}\text{.}\) If \(p|q_{n-1}\text{,}\) we're done by Claim 1. Otherwise, \(p|q_1 q_2 \cdots q_{n-2}\text{.}\)

Repeating this process, we see that either \(p|q_i\) for some \(i > 2\text{,}\) or \(p|q_1 q_2\text{.}\) Now the definition of a prime guarantees that \(p|q_1\) or \(p|q_2\text{,}\) in which case Claim 1 requires that \(p = q_1\) or \(p = q_2\text{.}\)

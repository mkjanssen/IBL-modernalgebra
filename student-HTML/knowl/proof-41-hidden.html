Let \(I\subseteq Z\) be an ideal. If \(I = \set{0}\text{,}\) then \(I = \ideal{0}\text{,}\) so suppose there is some nonzero \(x\in I\text{.}\) Define \(S = \setof{m\in \Z}{m > 0}\text{.}\) Note that \(S\ne \emptyset\text{,}\) as if \(m\in I\text{,}\) \((-1)m \in I\) also.
By WOP, \(S\) has a least element, call it \(d\text{.}\)
Claim: \(I = \ideal{d}\text{.}\)
It is clear that \(\ideal{d}\subseteq I\text{.}\) Now let \(x\in I\) be nonzero, and write \(x = dq + r\) using the division algorithm. Observe that \(0 \le r = x - dq \lt d\text{,}\) but as \(x\in I\) and \(-dq \in I\text{,}\) we must have \(r\in I\text{.}\) To avoid contradicting the WOP, we must have \(r = 0\text{.}\) Thus, \(x = dq\) and \(x\in \ideal{d}\text{.}\)

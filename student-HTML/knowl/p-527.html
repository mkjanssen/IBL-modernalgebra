Existence: If \(f(x) = 0\text{,}\) then \(q(x) = 0\) and \(r(x) = 0\) will do. If \(f(x)\ne 0\) and \(\deg f \lt \deg g\text{,}\) then \(q(x) = 0\) and \(r(x) = f(x)\) will suffice. Thus, we need only consider the case in which \(f(x) \ne 0\) and \(\deg f \ge \deg g\text{.}\) We use induction on \(\deg f = m\text{.}\)
